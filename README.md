# Hello, Nix!

A small assortment of `Hello, World!`-like exercises with the Nix package manager.

Needed things
---
  * [Nix!](https://nixos.org/nix/)
---
Inside a given subdirectory,
```bash
$ bash shell.sh
```
will run drop you into a `nix-shell` with a some assortment of libraries. Run the other `.sh` script(s) to test out the `nix-shell` environment. Once you're done, <kbd>ctrl</kbd> + <kbd>d</kbd> will set the world to rights.
